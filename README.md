# MoodMate – Emotion Detection using Machine Learning

## Project Overview

MoodMate is a machine learning project that detects emotions from text.
The model analyzes user input and predicts emotions such as happiness, sadness, anger, or fear.

## Technologies Used

* Python
* Machine Learning
* Scikit-learn
* Pandas
* Jupyter Notebook

## Dataset

The dataset contains text samples labeled with emotions used to train the model.

## Project Files

* Mood_Detection.ipynb – ML notebook
* emotion_dataset_raw.csv – training dataset
* model.py – emotion prediction model
* text_emotion.pkl – trained model
